## 4) Demo flow (কি কি দেখাবে, কোন order এ)

## Part A: Project overview (45 sec)

বলবে:
- এটা Python দিয়ে বানানো custom interpreter
- `.bnl` file পড়ে command execute করে
- Banglish keywords use করে

দেখাবে:
- `interpreter.py`
- `program.bnl`
- `README.md`

---

## Part B: Basic syntax demo (1.5 min)

`program.bnl` থেকে দেখাও:
- Variable: `dorlam x 10`
- Math: `jog`, `biyog`, `gun`, `vag`
- Print: `dekhao`
- Input: `naw`

Terminal এ run:

```powershell
program
```

Run করার পর output explain করো line by line।

---

## Part C: Condition + Loop demo (1.5 min)

`program.bnl`/`test.bnl` এ দেখাও:
- `jodi ... nahole ... shesh`
- `guraw ... shesh`
- `bhenge_jao`, `cholte_thako`

Terminal command:

```powershell
program test
```

বলবে:
- continue কোথায় skip করছে
- break কোথায় loop stop করছে

---
