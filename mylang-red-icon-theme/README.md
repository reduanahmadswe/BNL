# BNL Red Icon Theme

This local VS Code extension adds a custom red icon for `*.bnl` files.

## Use in VS Code

1. Open folder `mylang-red-icon-theme` in a VS Code window.
2. Press `F5` to launch Extension Development Host.
3. In the new window, open your workspace.
4. Run `Preferences: File Icon Theme`.
5. Select **BNL Red Icon Theme**.

After this, files ending with `.bnl` use the red icon.
