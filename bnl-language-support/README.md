# BNL Language Support

Custom syntax highlighting for `.bnl` (Banglish language) files.

- Highlights Banglish keywords: `dorlam`, `dekhao`, `guraw`, `shesh`
- Highlights word operators: `jog`, `biyog`, `gun`, `vag`
- Supports comments (`#`), numbers, strings, and identifiers
- No Python diagnostics for `.bnl` files
